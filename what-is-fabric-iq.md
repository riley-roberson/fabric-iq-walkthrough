# What is Fabric IQ (Preview)?

IQ is a workload for **unifying data across OneLake** and organizing it according to the language of your business. Data is exposed to analytics, AI agents, and applications with consistent semantic meaning and context.

Source: [What is Fabric IQ?](https://learn.microsoft.com/fabric/iq/overview)
{: .note }

---

## Items in the IQ workload

| Item | Purpose |
|:-----|:--------|
| **Ontology (preview)** | Enterprise vocabulary and semantic layer — entity types, properties, relationships, rules, constraints — bound to real data |
| **Plan (preview)** | Unified no-code platform for collaborative planning, reporting, analytics, data integration |
| **Graph (preview)** | Native graph storage and compute for nodes, edges, and traversals over connected data |
| **Data agent (preview)** | Conversational Q&A over enterprise data using generative AI |
| **Operations agent (preview)** | AI agent to monitor real-time data and recommend/trigger business actions |
| **Power BI semantic model** | Curated analytics model optimized for reporting with measures, hierarchies, DAX |

---

## How items work together

### Ontology + Semantic model
Generate or align Power BI semantic models so terminology and KPIs stay consistent across reports. Define enterprise concepts (Customer, Shipment, Breach) once.

### Ontology + Graph
Ontology declares what connects and why. Graph stores and computes traversals.

### Ontology + Data/Operations agents
Ontology grounds agents in shared business semantics and rules. Agents retrieve context, reason across domains, and recommend or trigger governed actions.

### Plan + Semantic model
Plan connects to existing semantic models, using dimensions and measures in planning sheets.

---

## Choosing the right item

| Item | When to use |
|:-----|:------------|
| **Ontology** | Cross-domain consistency, governance, AI/agent grounding, reasoning across processes |
| **Graph** | Relationship-heavy questions (impact chains, communities, shortest paths) dominate decision-making |
| **Power BI semantic model** | Trusted KPIs and fast visuals with dimensional modeling, DAX, governed self-service BI |
